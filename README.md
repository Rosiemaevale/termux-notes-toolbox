# Termux Notes Toolbox

A lightweight, offline-first CLI system for:
- frictionless note capture
- adaptive daily prompts
- habit signals and weekly summaries
- zero cloud or AI dependency

Designed for Termux on Android.

## Commands
- note, checkin
- review, summarize
- habits, weekreport
- exportweek

## Philosophy
One line per day counts.
No backfilling.
No punishment loops.
